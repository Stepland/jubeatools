# Jubeatools
A toolbox for jubeat file formats

## Conversion
jubeatools can handle conversions in the following way :

### Memon
|        | input | output |
|--------|:-----:|:------:|
| v0.2.0 | ✔️     | ✔️      |
| v0.1.0 | ✔️     | ✔️      |
| legacy | ✔️     | ✔️      |

### Jubeat Analyser
|                      | input | output |
|----------------------|:-----:|:------:|
| #memo                | ✔️     | ✔️      |
| mono-column (1列形式) | ✔️     | ✔️      |
