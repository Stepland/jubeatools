# Jubeatools
A toolbox for jubeat file formats

## Conversion
jubeatools supports the following formats :

### Memon
|        | input | output |
|--------|:-----:|:------:|
| v0.2.0 | ✔️     | ✔️      |
| v0.1.0 | ✔️     | ✔️      |
| legacy | ✔️     | ✔️      |

### Jubeat Analyser
|                      | input | output |
|----------------------|:-----:|:------:|
| #memo2               | ✔️     | ✔️      |
| #memo1               | ✔️     | ✔️      |
| #memo                | ✔️     | ✔️      |
| mono-column (1列形式) | ✔️     | ✔️      |
